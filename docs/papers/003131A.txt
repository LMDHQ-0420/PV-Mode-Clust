论文题目：A 题 光伏电站发电功率日前预测问题
            光伏电站发电功率日前预测问题


                    摘   要

  为了实现“碳中和、碳达峰”，实现节能减排，光伏发电站得到了广泛应用。
然而，由于新能源的波动明显，光伏发电功率预测仍是一大难题。本文结合时序
特征与 NWP 信息，建立 FCM-EMD-tSNE-RF 预测模型，通过划分气象场景、
分解 NWP 信息，在 10 个光伏电站的真实数据下，验证了该模型传统气象学模
型有效地提高了光伏发电功率的预测效果，并进一步验证了 NWP 降尺度在光伏
发电预测场景下的有效性。
  针对问题一，本文使用光伏电站地理位置信息，构建气象学模型，通过联合
使用 SPA 模型、Ineichen-Perez 模型、简单各向同性模型，实现了光伏电站发电
功率的预测，且预测 acc 值在 75%左右。
  针对问题二，本文使用 RF 模型，通过构建时序特征，有效地提升了光伏电
站发电功率的预测效果，其 acc 值在 89%左右。
  针对问题三，本文建立了 FCM-EMD-tSNE-RF 预测模型，使用 FCM 算法
将辐照度、温度、湿度对其进行聚类，并划分气候场景。对于每一个气候场景，
使用 EMD 提取气候特征的内在特征，并使用 t-SNE 对 EMD 分解结果进行降维，
确保特征空间不会过大，最后，融合处理后的 NWP 特征与时序特征，使用 RF
模型进行预测。结合各评估指标，分析融入 NWP 信息后预测精度能否有效提高，
并给出相应气候场景的划分方案。其 acc 值在 93%左右。
  针对问题四，本文使用 EPA5 数据集，通过对 EPA5 进行预处理，并结合二
维线性插值法获得更高精度的 NWP 数据，将 FCM-EMD-tSNE-RF 预测模型的
acc 值提高了 1%左右。


  关键词：随机森林；模糊 C 均值聚类；经验模态分解；t 分布随机邻域嵌入
1 问题重述
1.1 问题背景

  光伏发电能够利用半导体材料的光电效应，把太阳能直接转化为电能。多个
光伏发电单元共同作用形成的规模化发电设施即为光伏电站。
  光伏板表面接受到的太阳辐射总量决定了光伏电站的发电功率。由于不同季
节时太阳倾角发生变化，辐照强度也会产生长周期变化；由于云量、阴雨、雾霾
等气象因素，辐照强度会发生短周期变化。
  光伏电站接入电网后会使光伏电站发电功率产生波动，从而影响电网的功率
平衡和频率调节。精准预测光伏电站的发电功率可以帮助电力调度部门提前准备
调度决策，来保障电网的功率平衡与运行安全。
  光伏电站发电功率日前预测是进行预测未来 24 小时至 48 小时的发电功率。
但光伏电站上方的云量、阴雨、雾霾等气象因素是不确定的，因而导致光伏发电
功率无法准确预测。目前，提升光伏电站发电功率的预测精度是当前工程领域关
键技术问题。

1.2 问题重述

  对于问题一，基于各光伏电站的经纬度，计算太阳辐照并推算可发功率，分
别得出长周期和短周期可发功率特性。根据实际功率与理论可发功率的偏差，分
析光伏电站发电功率特性。
  对于问题二，基于历史功率的光伏电站，建立发电功率预测模型，对发电功
率进行预测，根据附件 1，分别计算各误差指标，分析预测模型的准确性。
  对于问题三，在问题二的基础上，融入数值天气预报（Numerical Weather
Prediction, NWP）信息，建立融入 NWP 信息的光伏电站日前发电功率预测模型，
得到结果并分析评价融入 NWP 信息能否有效提高预测精度，若能有效提高，则
给出提高预测精度的场景划分方案，并进行验证。
  对于问题四，由于传统气象预报空间分辨率尺度较大，通常在千米级别，而
MW 级光伏电站覆盖面积可能小于天气预报的空间尺度。在现有的 NWP 数据基
础上，通过机器学习、空间插值、统计模型等得到更小空间尺度的气象预报信息
（NWP 空间降尺度）可否提高光伏功率预测精度。请结合空间降尺度预测结果，
检验方法的可行性，并分析其原因。


2 问题分析

                      1
2.1 问题一的分析

   本题根据数据集中光伏电站的地理位置信息，可计算得到太阳直射点辐照度、
光伏板所在位置与太阳的相对位置（太阳高度角、方位角），进而结合光伏板倾
角，计算光伏板斜面的总辐照度，再结合光伏电站装机容量计算出可发功率。根
据可发功率和日内波动得出其长周期和短周期变化并进行分析，再计算其与实际
功率的偏差并进行分析得出结论。

2.2 问题二的分析

   本题要求基于数据集中除去第 2、5、8、11 个月的最后一周的可发功率构建
训练集，可采取数据驱动的方式，建立基于历史功率的光伏电站日前发电功率预
测模型，对第 2、5、8、11 个月的最后一周的可发功率进行预测。首先，对时间
序列数据进行特征提取，建立随机森林预测（Random Forests, RF）模型进行预
测，得到预测结果，并结合各评估指标，分析预测模型的是否准确。

2.3 问题三的分析

   本题在问题二的基础上，融入 NWP 信息，建立融入 NWP 信息的光伏电站
日前发电功率预测模型，对第 2、5、8、11 个月的最后一周的可发功率进行预测。
将辐照度、温度、湿度对其进行模糊 C 均值聚类（Fuzzy C-Means Clustering, FCM），
并划分气候场景。对于每一个气候场景，使用经验模态分解（Empirical Mode
Decomposition, EMD）提取气候特征的内在特征，并使用 t 分布随机邻域嵌入（t-
distributed Stochastic Neighbor Embedding, t-SNE）对 EMD 分解结果进行降维，
确保特征空间不会过大，最后，融合处理后的 NWP 特征与时序特征，使用 RF
模型进行预测。结合各评估指标，分析融入 NWP 信息后预测精度能否有效提高，
并给出相应气候场景的划分方案。

2.4 问题四的分析

   本题在问题三的基础上，对 NWP 数据进行降尺度，进而获得更高精度的
NWP 数据，并提升模型的预测效果。具体来说，本文使用 EPA5 数据集，通过对
EPA5 进行预处理，并结合二维线性插值法获得更高精度的 NWP 数据，提高了
模型的预测效果。

6.1 光伏电站可发功率的理论计算

  基于以上结论，本文可作以下合理猜测：
1) 由于夏季光伏电站的发电功率明显高于冬季，可以猜测光伏电站的发电功率
  与温度或辐照度相关；
2) 晴天情况下，辐照度值较大，可能导致可发功率较大；
3) 由于本文假设该地区均为晴天，故理论可发功率和实际发电功率的趋势相似。
  然而实际情况下，该地的天气情况可能略差，导致理论辐照度高于实际辐照
  度，因此，理论可发功率的普遍要大于实际发电功率。


7 基于历史功率的光伏电站日前发电功率预测模型
7.1 随机森林模型（RF）介绍

   随机森林[6]的基本组成单元是决策树，其基本思想是一种二分递归分割方法，
在计算中利用二叉树，在一定的分割规则下将样本集分割为两个子样本集，使得
生成的决策树的每个非叶节点都有两个分枝，在子样本上重复上述，直至不可再
分割为止。其流程图如图 5 所示。




                         8
             图 5 随机森林模型预测流程图
   RF 选取样本时为有放回随机选取，因此具有随机性。不同的树分别学到整
体的一部分特征，最终通过回归方法取平均值，得到最终的预测结果。由于单棵
决策树往往容易出现过拟合问题，因此需要通过聚集多个模型来提高预测精度。
RF 采用的是 Bagging 方法来组合决策树，其基本思想是：利用 Bootstrap 重抽样
方法从原始样本中抽取多个样本，对每个样本进行决策树建模，然后组合多棵决
策树的预测，以求出预测结果。
   对于一个决策树，使用信息增益准则来选取最佳划分点，假定 a 属性有 V 个
可能的取值 {a1 , a 2 , …, aV } ，则有公式如下：

8.1 相关理论基础

①模糊 C 均值（FCM）算法


                                     11
   FCM 聚类算法通过将数据点与各聚类中心的相似度总和作为优化目标，通
过优化过程最小化该目标函数以确定聚类中心[7]。其目标函数实际上代表了所有
数据点到各聚类中心的欧氏距离之和（即误差平方和）。聚类的过程就是最小化
目标函数的过程，通过反复的选代运算，逐步降低目标函数的误差值，当目标函
数收敛时，可得到最终的聚类结果。具体来说，其目标函数如下：

其中，C 为聚类中心，隶属度矩阵 U 表示数据点属于某个聚类的程度，FCM 把
n 个向量 xi ( i = 1, 2...n ) 分为 c 个模糊组，并求每组的聚类中心，使得非相似性指
标的价值函数达到最小。
②经验模态分解（EMD）
   EMD 分解是对非平稳信号进行平稳化处理的一种手段[8]。其结果是将信号
中不同尺度的波动和趋势进行逐级分解，产生一系列具有不同特征尺度的数据序
列，每一个序列称为一个本征模态函数（Intrinsic Mode Function, IMF）
                                              。对于一
个 IMF，其必须满足以下两个条件：在整个时间范围内，局部极值点和零交叉点
的数量必须相等或差值最多为 1；由局部极大值和局部极小值形成的两个包络曲
线的平均值在任意点上应为 0。
③t-SNE 降维
   t-SNE 降维是一个常用的高维数据降维方法，它通过概率分布将高维空间中
相似的数据点映射到低维空间中的相近位置，同时保持数据点间的概率关系[9]。
由于 t-SNE 主要利用 t-分布来增强低维空间中数据点的分离度，因此其特别适合
于捕捉数据的局部结构，且能揭示复杂的、非线性的数据分布，从而避免了在低
维空间中常见的拥挤问题。

8.2 FCM-EMD-tSNE-RF 预测模型

①使用 FCM 划分气象场景
   由于光伏电站的发电功率对气象条件敏感，尤其对辐照度、温度、湿度等气
象数据敏感，因此，本文考虑对气象场景进行划分，对每一个场景分别使用 RF
模型进行预测。具体来说，本文使用 FCM 聚类算法对辐照度、温度、湿度进行
聚类，将其划分为 4 个簇，分别对应 4 类气象场景。
   在本文中，使用 FCM 聚类而非常见的 K-means 聚类，主要有以下原因：
1) 因为天气现象通常具有渐变和重叠的特性，其边界较为模糊。而 FCM 是一

  量可能在不同的天气模式中表现出相似的特征。由于不强制要求每一个数
  据点都严格属于一个聚类，因此，其能够更好地处理这种不确定性。

②使用 EMD 分解气象数据
  由于气象数据多为非线性且非平稳的，故其统计特性随时间变化明显，然而，
气象数据的特征缺由于其变化较大而难以直接提取。EMD 可以自适应地将复杂
的气象数据分解为若干个 IMFs，其代表了原始数据的不同内在振荡模式，这有
助于提取其重要特征，为 RF 模型提供更有意义的输入。
  此外，在分解过程中，EMD 可以将噪声和非相关成分从原始数据中提取出
来，这有助于 RF 模型更准确地捕捉到数据的潜在规律。
③使用 t-SNE 对 EMD 的分解结果进行降维
  EMD 分解虽然能够提取原始数据中的显著特征，但是其可能会产生多个
IMFs，每个 IMF 都包含原始数据的一部分信息。直接将所有 IMFs 作为特征输入
模型可能导致特征空间过大，增加模型训练和预测的计算复杂度。因此，需要对
EMD 的分解结果进行降维。
  由于气象数据通常包含复杂的非线性关系，而 t-SNE 作为一种非线性降维技
术，能够很好地捕获气象数据的局部结构，因此，本文使用 t-SNE 对 EMD 的分
解结果进行降维。
④分别使用 RF 对不同气象场景的数据进行预测
  使用 FCM 划分气象场景后，可以对每一个气象场景分别训练独立的 RF 模

          parts = col.split('_')
          if len(parts) == 3:
               lat = float(parts[1])
               lon = float(parts[2])
               lats.add(lat)
               lons.add(lon)
               lat_lon_pairs.append((lat, lon))


     # 转换为排序后的列表
     lats = sorted(list(lats))
     lons = sorted(list(lons))


     return lats, lons, lat_lon_pairs




def create_interpolator(df, variable, lats, lons):

     grid_data = np.zeros((len(lats), len(lons)))


     # 填充网格数据

               col_name = f"{variable}_{lat:.2f}_{lon:.2f}"
